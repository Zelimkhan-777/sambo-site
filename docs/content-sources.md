- биографии;
- спортивные звания;
- достижения;
- результаты соревнований;
- даты мероприятий;
- статистику;
- документы;
- гранты;
- контактные данные;
- подписи к фотографиям.

Для каждого материала по возможности сохранять:

- название источника;
- ссылку;
- дату публикации;
- дату события;
- статус проверки.

Рекомендуемые статусы:

```text
verified
requires-review
placeholder
```

## Правила работы с фотографиями

В первую очередь использовать фотографии:

- с официальных страниц федерации;
- с официальных страниц спортивных школ;
- с сайта Всероссийской федерации самбо;
- из официальных публикаций организаторов соревнований;
- предоставленные представителями федерации.

Не использовать случайные изображения из поисковой выдачи без проверки источника.

Не указывать имя человека на фотографии, если оно не подтверждено подписью официального источника.

Для найденных изображений сохранять:

- исходную ссылку;
- источник;
- описание события;
- предполагаемый раздел сайта;
- статус разрешения на использование.

Рекомендуемый статус для найденных в интернете фотографий:

```text
usageStatus: "requires-approval"
```

До подтверждения разрешения такие изображения считаются временными.
